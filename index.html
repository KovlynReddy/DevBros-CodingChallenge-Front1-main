How I Tackled this

  1. Requires running from a server to avoid CORS policy error on fethcing data
  2. Suggested using VS Code and downloading Live Server to run
  3. Pulls data without knowing its 2D shape , traverses JSON first Set of Children and constructs table off the Keys of the First Children
  4. Base search and sort algorithms taken from W3 School and changed to take in an index for the coloumns
  5. Attempted to use clicking of table header to trigger the sorting , tried a couple different solutions such as vanilla eventhandler and jquery event handler
  6. However I am coding 2 other assessments this weekend and decided to go with this crude solution of a column input
  7. I would preffered to use Datatables but it was strongly advised against
  8. I chose to use vanilla html css and js instead of my main C# .net stack
  9. I am a Full Stack Developer but my comfort is defnitly in the back end
